RegionMapper v1.0.0

Define. Map. Automate.

A lightweight precision mapping tool for defining screen regions in real time.

FEATURES
- One Point & Four Point mapping
- Live draggable overlay
- Instant copy (F4)
- Persistent history
- Clean minimal UI

HOTKEYS
F2 - Toggle tool
F4 - Copy coordinates
ESC - Back

NOTES
- Works across windows (even when switching apps)
- Requires assets/images folder for UI elements
