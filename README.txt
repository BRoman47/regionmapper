## 🚀 Features

* 🎯 One Point mapping
* 🔲 Four Point mapping
* ⚡ Live draggable overlay
* 📋 Instant copy (F4)
* 🕘 Persistent history
* 🖤 Clean minimal UI

---

## 🎮 Hotkeys

| Key | Action           |
| --- | ---------------- |
| F2  | Toggle tool      |
| F4  | Copy coordinates |
| ESC | Back             |

## ⚙️ How to use

1. Run the application
2. Press **F2** to activate
3. Move mouse to desired position
4. Press **F4** to copy coordinates
